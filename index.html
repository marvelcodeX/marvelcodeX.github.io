03

RAG + Local LLM

PDF Intelligence System

A lightweight Retrieval-Augmented Generation (RAG) system that lets you query any PDF using fully local LLMs — no API keys, no cloud, no cost.

Ask natural language questions about documents and get grounded answers powered by FAISS vector search and Sentence Transformers, all running locally via Ollama.

  • Python
  • Streamlit
  • FAISS
  • Ollama
  • Sentence Transformers
  • PyMuPDF
  • Gradio
GitHub
More ↓

Built to be 100% local and private — designed to work with lightweight models under 2GB like Phi-3, qwen2:1.5b, and tinyllama. Privacy-first document intelligence.
